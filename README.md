## 项目目录说明

url-tools/
├── .vitepress/             # VitePress 核心配置目录
│   ├── theme/              # 自定义主题和样式
│   │   ├── index.ts        # 主题入口文件 (加载自定义样式)
│   │   └── style.css       # 自定义 CSS (如主页渐变色)
│   └── config.mts          # 网站主配置文件 (导航栏、侧边栏、页脚、编辑链接等)
├── public/                 # 静态资源目录 (图片、Logo、Favicon 等)
│   └── images/             # 存放网站图片资源
│       └── logo.png        # 网站 Logo 或主页图片
├── sites/                  # 网站内容目录 (所有网站列表的 Markdown 文件)
│   ├── index.md            # 网站列表总览页 (卡片导航)
│   ├── resource.md         # 素材与资源分类
│   ├── music.md            # 音乐学习分类
│   ├── entertainment.md    # 休闲娱乐分类
│   ├── tools.md            # 在线工具分类
│   └── personal.md         # 个人与推荐分类
├── about.md                # 关于本站页面
├── announcements.md        # 公告发布页面
├── contributors.md         # 贡献者名单页面
├── help-us.md              # 贡献指南页面 (欢迎大家 PR)
├── index.md                # 网站主页 (Home Page)
├── LICENSE                 # MIT 开源许可文件
├── package.json            # 项目依赖和脚本定义文件 (如果存在的话，npm install 依赖此文件)
└── README.md               # 项目说明文件 (你现在正在看的文件)
